# 项目名称：Mini-mango问答系统
## 功能简介：
* 实现登录（需要后台验证）和注册功能
* 主页面有问题列表（问题排序可根据发布时间，浏览次数排序）
* 提问页面（需要设置悬赏）
* 回答页面
* 用户个人信息页面
